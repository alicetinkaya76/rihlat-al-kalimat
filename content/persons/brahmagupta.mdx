      en: |
        Try to withdraw 0 from an ATM and the transaction is refused —
        because the machine handles zero as a *quantity*, and there is an
        answer to "what is zero subtracted from this": no change. The same
        holds for 0 × 1,000,000: the answer is 0, a number, not "nothing".
        This simple principle — that zero can be operated on, alongside
        other numbers — is the spine of modern arithmetic, but until the
        early seventh century it was not formally written as a rule in
        any mathematical tradition on earth. The Babylonian zero was a
        *placeholder mark* (to separate, say, 207 from 27); the Greek
        astronomer Ptolemy used a *little circle*; but none of these said
        "zero is a number that obeys the following rules." That was done
        by Brahmagupta. Chapter 18 of his *Brāhmasphuṭasiddhānta*, written
        in 628, sets the rules out explicitly: *zero plus zero is zero; a
        number plus zero is the number; a number times zero is zero*. The
        sentences read like the modern textbook — because the modern
        textbook is, by indirect descent, copied from there.

        languages, one logic.
